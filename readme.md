#介绍

* 起源于一次在命令行显示嵌入式系统参数的尝试
* 方便在CLI中，对表格类数据进行打印输出
* 方便对二维任意数据类型进行管理

# 功能

* 创建静态表格，初始化行数与列数
* 支持表格左 右 中对齐功能
* 表格支持字符串
* 表格CLI打印
* 将csv格式字符串转化为表格数据
* 可自由设置表格边框，也可从支持的形式中进行选择
